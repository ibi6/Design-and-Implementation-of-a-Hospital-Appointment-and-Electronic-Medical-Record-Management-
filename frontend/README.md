# 慧医通 · 前端展示版

基于 React + Vite + Tailwind 的医院预约挂号与电子病历管理系统前端（Mock 数据）。

## 启动

```bash
cd frontend
npm install
npm run dev
```

访问：http://localhost:5173

## 已完成页面

- 落地首页、登录、注册、404
- 患者：工作台、科室、医生、预约、病历、个人中心
- 医生：工作台、接诊、写病历、排班、历史病历
- 管理：看板、科室/医生/排班/预约/用户管理

## Mock 说明

- 鉴权与业务数据均在浏览器 localStorage
- 刷新页面数据会保留；可在患者端「个人中心」重置演示数据
- 尚未接入 SpringBoot 后端
